# MiSubtitles
Экранные субтитры как в игре MiSide<br>
miside_letters.pyw - создать анимацию<br>

misubs_client.py - клиент для отображения субтитров<br>

misubs_server_vosk.py - сервер для распознавания речи на Vosk (! нужно распаковать нужную модель в папку model)<br>

misubs_server_whisper.py - сервер для распознавания речи на OpenAi whisper<br>
