# MiSubtitles
**Дисклеймер<br>**
> Я не разработчик Miside, все права на оригинальную игру принадлежат AiHasto.<br>

Экранные субтитры как в игре MiSide<br>
miside_letters.pyw - создать анимацию без голоса<br>

misubs_client.py - клиент для отображения субтитров<br>

misubs_server_vosk.py - сервер для распознавания речи на Vosk<br>

misubs_server_whisper.py - сервер для распознавания речи на OpenAi whisper<br>
